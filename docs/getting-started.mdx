# Getting Started

PUDA is a hardware harness for Physical AI. Read more about it at [puda.co/blog/whitepaper](https://www.puda.co/blog/whitepaper).

## What You Will Need

Some experience with Linux and Docker, plus hosts for the AI agent, NATS server, and any machine edge services.

See [Hardware Requirements](./setup/hardware-requirements) for recommended minimums.
