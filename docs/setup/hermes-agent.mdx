## Telegram Group Privacy Mode

If you connect Hermes to a Telegram group, configure Telegram group privacy mode so Hermes can receive the messages it needs. Follow the [Hermes Telegram messaging guide](https://github.com/NousResearch/hermes-agent/blob/main/website/docs/user-guide/messaging/telegram.md).

Next, [install the PUDA CLI and skills](./puda-cli-and-skills) on your machine so Hermes can start using PUDA.
